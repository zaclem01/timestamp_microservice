Timeserver Microservice API: FCC project

Instructions:

  • Add a unix timestamp or natural language date to the url
  • The API will return a JSON object converting the input to a unix timestamp and natural language date
  • If the input is incorrect, the JSON object keys will be 
    null

Examples:

  • https://timestamp-service-fcc2089.herokuapp.com/jan%201%2016
    yields: 
    {unix: 1451606400,natural: "January 1, 2016"}
  • https://timestamp-service-fcc2089.herokuapp.com/1451606400
    yields: 
    {unix: 1451606400,natural: "January 1, 2016"}
  • https://timestamp-service-fcc2089.herokuapp.com/ksdajklfjd
    yields: 
    {unix: null,natural: null}
